# Static App

Demonstrates proper asset organization in a Weblet.

## Features

- Organized `/assets` directory
- External CSS stylesheet
- SVG logo and icons
- Responsive design

## Structure

```
static-app/
  APP.md
  index.html
  assets/
    style.css
    logo.svg
    images/
      hero.svg
```

## Running

```bash
weblet run .
```

Or open index.html directly in a browser.
